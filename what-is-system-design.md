# What is system design?

System is the process of designing products and services. It invloves recognizing the 
  - Architecture
  - Components (e.g Backend, Frontend, Database)
  - Modules (e.g Auth, User, Product, Order, Payment)

Building the best system with minimal resources (cost and services) and time is the goal of system design.

## Importance of SD
1. Re-wires your brain to solve problems in a very structured manner.


## Process of System Design
1. Break down on problem statement into sub-problems.
2. Decide the key Components and Modules
3. Decide on the boundaries of each component
4. Revisit design for scalability and performance.
5. Check if the design is Fault-Tolerant and available.
